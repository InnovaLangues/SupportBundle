# SupportBundle

## Installation

Install with composer :

    $ composer require "innova/support-bundle" "1.*"
    $ php app/console claroline:plugin:install InnovaSupportBundle

## Uninstall 

    $ php app/console claroline:plugin:uninstall InnovaSupportBundle
